## Prerequisites:
- Node version 6.\*
- npm version 3.\* (comes with Node 6.\*)
- git


## Project Setup:
Use a command prompt with admin privileges
```
git clone https://github.com/tigermarques/dateformat-converter.git
cd dateformat-converter
npm run setup
```
